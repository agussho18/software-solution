# Software Solution Company Profile

Website company profile statis untuk Software Solution yang siap diunggah ke GitHub dan Netlify.

## Struktur file
- index.html: halaman utama
- styles.css: styling website
- script.js: script sederhana untuk footer tahun

## Deploy ke Netlify
1. Upload folder ini ke GitHub.
2. Buka Netlify, pilih New site from Git.
3. Hubungkan repository Anda.
4. Publish site.

Website ini menggunakan HTML, CSS, dan JavaScript murni sehingga tidak memerlukan build step.
